Convert [WP Geo](http://wordpress.org/extend/plugins/wp-geo/) data to [Geo Mashup](http://wordpress.org/extend/plugins/geo-mashup/) data. 

== Description ==

Simple data conversion tool. Modifies the database, so back it up first!  
Geo Mashup can optionally add data in [GeoData format](http://codex.wordpress.org/Geodata) also.
