Convert [WP Geo](http://wordpress.org/extend/plugins/wp-geo/) and other custom field coordinate data to [Geo Mashup](http://wordpress.org/extend/plugins/geo-mashup/) data. 

== Description ==

Simple data conversion tool. Modifies the database, so back it up first!  
Should work for latitude and longitude data stored in independent custom fields, like
the [GeoData format](http://codex.wordpress.org/Geodata), or a single fields with comma 
separated latitude and longitude decimal values.
